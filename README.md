## 📌 Description

**BLUEBOX** is a **12-in-1 multi-tool** for developers and security enthusiasts. It combines the power of **AI coding assistance**, **web cloning**, **file management**, **command execution**, and **tool installation** into one professional Termux/Linux tool.

## ✨ Features

| # | Feature | Description |
|---|---------|-------------|
| 1 | 🌐 **Web Cloner** | Clone any website (HTML + CSS + JS) |
| 2 | 💬 **Code Chat** | Chat with AI (Gemini) about coding |
| 3 | 💻 **Generate Code** | Generate code in Python/JS/Bash/Go |
| 4 | 📖 **Explain Code** | Get AI explanation of any code file |
| 5 | 🐛 **Debug Code** | AI-assisted debugging |

| 10 | 💬 **AI Terminal** | Conversational AI terminal |
